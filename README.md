# ArtisanConnect Ghana

ArtisanConnect Ghana is an AI-powered platform designed to bridge the gap between customers and skilled artisans in Ghana. It features intelligent matching, identity verification, location-aware discovery, and trust-based reputation mechanisms.

## Tech Stack

- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS, ShadCN UI
- **Backend:** Node.js, Express.js, TypeScript, Socket.io
- **Database:** PostgreSQL (Supabase) + pgvector
- **AI Layer:** Groq API (llama-3.3-70b-versatile, mixtral-8x7b-32768)
- **Maps:** Mapbox GL JS
- **Email:** Resend
